# My Portfolio

A modern, responsive portfolio website showcasing my projects, skills, certificates and achievements.

## 🔍 Overview

This portfolio is built to present my work in a clean and interactive way, highlighting real-world projects, technical skills, and development experience. It focuses on performance, responsiveness, and modern UI/UX design.

## ✨ Features

* Responsive design for all devices
* Project showcase with both live demo and github repository
* Skills section with categorized technologies
* Achievements & certifications display
* Contact section for easy communication

## 🛠️ Tech Stack

* **Frontend:** React, TypeScript
* **Styling:** Tailwind CSS
* **Tools:** CloudFlare (deployment)

## 📁 Sections Included

* Home
* About
* Projects
* Skills
* Certifications
* Achievements
* Contact
